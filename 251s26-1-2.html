1.2: Integrals as General and Particular Solutions

Some special cases of first order ODE

The most general form of a first-order ODE is $$\frac{dy}{dx} = f(x,y).$$ The special case (where the independent variable doesn’t appear in the ODE) $$\frac{dy}{dx} = f(y)$$ is called autonomous — the “law of nature" (the ODE) doesn’t depend on the independent variable. For many phenomena, we first consider the simpler case with these “static laws of nature" since they’re simpler to analyze 1, before considering a refined model of the form $\frac{dy}{dx} = f(x,y).$

Annotations from class

Newton’s cooling $\dfrac{dT}{dt} = k(A-T)$, the simple population model $\dfrac{dP}{dt} = kP$, and the logistic population model $\dfrac{dP}{dt} = kP(M-P)$ are all examples of autonomous first order ODE.

The “opposite" special case would be $$\frac{dy}{dx} = f(x),$$ where the differential equation doesn’t actually involve the unknown function itself.
While technically still a differential equation, it’s “easy" in the sense that it can be solved for y(x) by simply antidifferentiating f(x): y(x) = ∫f(x) dx + C.

Example. Consider $\frac{dy}{dx} = e^{2x} + x^2$.

  1. Find the general solution.

  2. Find the particular solution satisfying y(1) = 3.

  3. Take a look on Desmos!

Annotations from class

(a) We have $$y = \int \dfrac 1 2 e^{2x} + \frac{x^3}{3} + C.$$ (b) Using y(1) = 3 leads to $$3 = \frac 1 2 e^{2} + \frac 1 3 + C \implies C = 3 - \frac 1 3 - \frac 1 2 e^2 = \frac 8 3 - \frac 1 2 e^{2},$$ and so the particular solution is then $$y = \frac 1 2 e^{2x} + \dfrac{x^3}{3} + \frac 8 3 - \frac 1 2 e^{2}.$$ (c) Plotting the above in Desmos will give you a slider for C, and adjusting the slider to $C = \frac 8 3 - \frac 1 2 e^2 \approx -1$ shows this choice of C makes y(1) = 3.

This “solvable by directly integrating" also works for equations of the following forms: $$\begin{aligned} \frac{d^2y}{dx^2} &= f(x) , \\ \frac{d^3y}{dx^3} &= f(x) , \\ &\vdots\\ \frac{d^ny}{dx^n} &= f(x) . \end{aligned}$$ We obtain one integration constant per derivative we undo, and so we obtain an n-parameter family for the nth-order “antiderivative equation".

Annotations from class

For example, the general solution to y‴ = ex/2 + sin (2x) + 6 is obtained by integrating three times: $$\begin{aligned} y'' &= 2e^{x/2} - \frac 1 2 \cos(2x) + 6x + C, \\ y' &= 4e^{x/2} - \frac 1 4 \sin(2x) + 3x^2 + Cx + D,\\ y &= 8e^{x/2} - \frac 1 8 \cos(2x) + x^3 + \frac{C}{2}x^2 + Dx + E. \end{aligned}$$ For the integral ∫ex/2 dx, we substitute u = x/2, du = dx/2 and so dx = 2 du. This leads to $$\int e^u \cdot \frac 1 2 \, du = \frac 1 2 e^u = \frac 1 2 e^{x/2}.$$

Similarly, for ∫sin (2x) dx we substitute u = 2x, du = 2 dx and so $dx = \frac 1 2 \, du$ and $$\int \sin(u) \cdot \frac 1 2 \, du = - \frac 1 2 \cos(u) = -\frac 1 2 \cos(2x),$$ since we remember $\frac{d}{dx} \cos x = - \sin x.$

We can absorb the factor of $\frac 1 2$ into the arbitrary constant C, and we also can use whichever names we want for the constants. So, we have the equivalent three-parameter family of functions $$y = 8 e^{x/2} - \frac 1 8 \cos(2x) + x^3 + c_1 x^2 + c_2 x + c_3,$$ and remember that we expect three arbitrary constants in the general solution of a third order ODE.

Position, velocity, and acceleration

We consider a particle moving along the x-axis with position/displacement given by x = f(t). Its velocity is $$v = \frac{dx}{dt} = f'(t),$$ and its speed is |v|. Its acceleration is $$a = \frac{dv}{dt} = \frac{d^2x}{dt^2} = f''(t).$$ If m is the mass of the particle, Newton’s second law states ma(t) = F(t) where F(t) is the force acting on the particle along its line of motion.

Annotations from class

Note that v is signed, and the sign indicates the direction of motion. If the x-axis is pointing to the right, then v > 0 means the particle is moving to the right, while v < 0 means the particle is moving to the left. The speed is always positive (or zero) — think about how your car speedometer doesn’t have negative values, or care in which direction you’re driving. Note that in the case of multidimensional motion, the velocity is a vector quantity and the speed is it’s length.

Constant acceleration

In the special case where the acceleration doesn’t depend on time: a(t) = a, we can integrate to obtain v(t) = at + v0 where v0 = v(0) is the initial velocity. We can integrate again to obtain $$x(t) = \frac 1 2 a t^2 + v_0t + x_0,$$ where x0 = x(0) is the initial position.

We can already do some rocket science...

A lunar lander is falling freely toward the surface of the moon at a speed of 450 m/s.
Its thrusters cause the lander to decelerate at 2.5 m/s2 (after the moon’s gravitational acceleration is accounted for).
At what height above the surface should the thrusters be activated to ensure a “soft touchdown" (v = 0 on impact)?

Annotations from class

Let x be the height above the surface of the moon, and suppose we turn the thrusters on at t = 0. Then, x(0) = x0 is the height we wish to solve for, and v(0) = −450. We have $a(t) = \frac{dv}{dt} = 2.5$ for all t ≥ 0. Integrating, we obtain v(t) = 2.5t + C, and using v(0) = −450 yields C = −450 and so $$v(t) =\frac{dx}{dt}= 2.5t - 450.$$ Integrating again, we have x(t) = 1.25t2 − 450t + C, and using x(0) = x0 yields C = x0 and so x(t) = 1.25t2 − 450t + x0. Let’s call T the landing time, so that x(T) = 0. To be a soft landing, we also know v(T) = 0 and so we solve 0 = 2.5T − 450 ⟹ T = 180. Then, x(T) = x(180) = 0 in our position function yields 0 = 1.25(1802) − 450(180) + x0 ⟹ x0 = 450(180) − 1.25(1802) = 40500 m. Therefore, we should activate the thrusters when the lander is 40.5 km from the moon’s surface.

Units


The weight of an object is the force exerted by gravity on the object, so at sea level on Earth is mg.

Annotations from class

When at the surface of earth (or nearby so that the gravitational acceleration can be assumed to be constant), we have that g is the constant of proportionality between an objects mass and weight.
For example, an object with mass 1 kg has weight 9.8 N in SI units.
Similarly, an object with mass 1 slug has weight 32 lb in imperial units.

Example. A projectile is fired straight upward with an initial velocity of 100 m/s from the top of a building 20 m high and falls to the ground at the base of the building. Find

  1. its maximum height above the ground

  2. when it passes the top of the building

  3. its total time in the air

Annotations from class

Let x(t) be the height of the projectile (in meters) above the ground at time t (in seconds).
We have $a(t) = \frac{dv}{dt} = -9.8$ for t ≥ 0, along with v(0) = 100, x(0) = 20.
Integrating yields v(t) = −9.8t + C, and using v(0) = 100 leads to v(t) = −9.8t + 100. Integrating again yields x(t) = −4.9t2 + 100t + C, and using x(0) = 20 yields x(t) = −4.9t2 + 100t + 20.

(a) To find the maximum of x(t), we set x′(t) = v(t) = 0 and find this occurs for $t = \frac{100}{9.8}.$ We see that v(t) is going from positive to negative at this t-value (it’s a linear function with negative slope), and so this must correspond to the absolute maximum of x(t). Therefore, the maximum height is $x\left( \frac{100}{9.8} \right) = \cdots.$
(b) We set x(t) = 20 and solve for t to determine when the projectile passes the top of the building. This yields 20 = −4.9t2 + 100t + 20 ⟹ 0 = t(−4.9t + 100).

We know x(0) = 20 already, and so we’re interested in the other solution $t = \frac{100}{4.9}.$ (c) We set x(t) = 0 and solve for t. Using the quadratic formula will yield one positive and one negative solution, and we will choose the positive solution since t < 0 is before the projectile is fired (“before our experiment begins").
