1.1: Differential Equations and Mathematical Models

Most interesting natural phenomena involve change, and the laws governing these phenomena are written in mathematical equations relating changing quantities.
An equation relating an unknown function and one or more of its derivatives is called a differential equation.

Examples of differential equations

  • The DE $$\frac{dy}{dx} = y^2 + x^2$$ involves the unknown function y(x) and its derivative $y'(x)=\frac{dy}{dx}$.

Examples of differential equations

  • The DE $$m\frac{d^2y}{dt^2} + b \frac{dy}{dt} + ky = 0$$ involves the unknown function y(t) and its first and second derivatives y′(t) and y″(t).

Annotations from class

This is the equation for a mass-spring-dashpot system, which is one type of damped linear oscillator. It is also the same equation as an LRC-circuit.
Here, y(t) is the displacement (from equilibrium) of a mass m attached to a spring. A spring has an equilibrium position (corresponds to it’s “preferred length"). If y > 0 then the spring is stretched, and if y < 0 the spring is compressed. You may recall from Physics 211 that a linear spring satisfies Hooke’s law, meaning the restoring force is proportional to the displacement from equilibrium, and the constant of proportionality k is called the spring constant, which measures the “stiffness" of the spring.
We also have a dashpot/shock absorber attached to the mass that provides a resistive (friction) force, and the dashpot has damping constant b.

We’ll study this equation in detail in section 3.4, and will consider the equation with a sinusoidal driving force in 3.6. In 3.6, we’ll explore the concept of resonance, which is when there are large amplitude oscillations for certain driving frequencies.

Examples of differential equations

  • The DE $$\frac{\partial u}{\partial t } = k \frac{\partial^2 u}{\partial x^2}$$ involves the unknown function u(x, t) and its partial derivatives $\frac{\partial u}{\partial t}$ and $\frac{\partial^2 u}{\partial x^2}.$

Annotations from class

This is the heat/diffusion equation, and models heat transfer via conduction along a thin 1-dimensional rod occupying 0 ≤ x ≤ L. Here, the unknown function u(x, t) is the temperature at position x and time t along the rod. k is the thermal diffusivity of the rod, and depends on what it is made of. For example, copper has a higher thermal diffusivity than steel, which is why expensive multiply stainless steel cookware has layer(s) of copper in between the steel so that it heats more evenly more quickly.
When considering functions of multiple independent variables, we consider partial derivatives and use the partial derivative notation with “curly d" to emphasize there are other independent variable(s) being held constant. In this case for example, $\frac{\partial u}{\partial t}$ is the partial derivative of temperature with respect to time, while the position x is held constant.

We’ll study the heat equation in Chapter 9 (along with the wave equation) using Fourier series.

Some vocabulary

The order is the order of the highest derivative appearing in the equation. An nth order ODE for y(x) takes the form F(x, y, y′, y″, ..., y(n)) = 0.
An ordinary differential equation (ODE) involves an unknown function of one input/independent variable.
A partial differential equation (PDE) involves an unknown function of multiple inputs/independent variables.
We’ll study ODE for most of the course, and then use Fourier series to solve two PDE towards the end (the heat equation and wave equation).

Annotations from class

The earlier example $\frac{dy}{dx} = y^2 +x^2$ is first order ODE.
The DE my″ + by′ + ky = 0 is a second order ODE.
The DE $\frac{\partial u}{\partial t} = k \frac{\partial^2 u}{\partial x^2}$ is a second order PDE.

An autonomous ordinary differential equation has no explicit dependence on the independent variable. The “laws of nature” (the DE) are static in “time" (the input we often have in mind for ODE).
A differential equation will often have physical constants/ physical parameters in addition to the independent and dependent variables — we will discuss how useful units are in helping is formulate models!

Annotations from class

The earlier example $\frac{dy}{dx} = y^2 +x^2$ is not autonomous, since the independent variable x appears in the equation. There aren’t any parameters in this equation.
The DE my″ + by′ + ky = 0 is autonomous, since the independent variable t does not appear in the equation. The parameters are the physical constants m, b, k.

Checking/verifying solutions

There is a difference between checking or verifying whether an equation is satisfied versus solving it.
To check/verify whether x = 3 is a solution of 2x + 5 = 11, we substitute to see: $$2(3) + 5 \overset{?}{=} 11, \qquad 11 \overset{\checkmark}{=} 11.$$ Solving would be doing $$\begin{aligned} 2x+5 &= 11, \\ 2x &= 6, \\ x &= 3. \end{aligned}$$ “Solving" typically means a systematic way to “undo" the equation and obtain the solution “from scratch".

Some more vocab

Example. Verify that y(x) = Cex2 satisfies/ solves the DE $$\frac{dy}{dx} = 2xy$$ for any constant C.
We say that, for example, y(x) = 5ex2 is a particular solution while y(x) = Cex2 is the general solution. The general solution will contain integration constant(s).

Annotations from class

For y(x) to be a solution of a DE, it must satisfy the DE for either all x, or at least for all x in some interval. (That is, only working for discrete values such as x = 0 and x = 7 would not be enough to make it a solution.)
Here, if y(x) = Cex2 we can compute $\frac{dy}{dx} = Ce^{x^2} \cdot 2x$. Substituting into the DE yields $$Ce^{x^2} \cdot 2x \overset{?}{=} 2x \cdot Ce^{x^2}, \qquad 2Cxe^{x^2} \overset{\checkmark}{=} 2Cxe^{x^2}$$ is true for all x, and so y(x) = Cex2 solves the ODE for any value of the constant C.

We’ll spend most of this course studying ordinary differential equations and systems of ordinary differential equations. We’ll study various calculus, algebraic, linear algebra, and “complex continuous frequency analysis" (Laplace transform) techniques, oftentimes learning several ways to solve certain equations.
When we study the heat and wave PDE towards the end, we’ll be using a “discrete frequency analysis" technique (Fourier series).

Basic goals of applied math

  1. Discover the differential equation that describes a specified physical situation.

  2. Find the appropriate solution of that equation (either exactly or approximately).

  3. Interpret the solution in the applied context.

Newton’s law of cooling

A small object with variable temperature T(t) (as a function of time t) sits inside a large surrounding medium maintained at a constant temperature A. The rate of change of T(t) is proportional to the temperature difference between the object and the surrounding medium. That is, $$\frac{dT}{dt} = k (A- T)$$ for some constant k > 0.

Annotations from class

Notice how the ODE/law of nature is described in words — “the rate of change of temperature is proportional to the temperature difference". Remember, we say y is proportional to x when y = kx for some constant of proportionality k. The temperature difference in this case is A − T or T − A, and if we want k > 0 we have $\dfrac{dT}{dt} = k(A-T).$

Annotations from class

Note that the equation as written is consistent with what we expect — if the object is colder than the surroundings (so that T < A), then A − T > 0 and the ODE states that $\dfrac{dT}{dt} > 0$ as expected. (Remember from Calc 1, when $\dfrac{dT}{dt} > 0$ we have the temperature T(t) is increasing, and we expect colder objects to heat up).
We could write the equivalent model $$\dfrac{dT}{dt} = k(T-A), \qquad k<0,$$ but whenever possible we try to make our parameters positive and include explicit minus signs as a reminder of the sign.

Simple population growth/decay

The time rate of change of P(t), the population at time t, is proportional to the population: $$\frac{dP}{dt} = kP$$ for some constant k. It turns out the general solution is given by P(t) = Cekt, which you can easily verify.

Example. Knowing that the general solution is P(t) = Cekt, suppose that the population took one hour to double from its initial value (at t = 0) P(0) = 1000. Find the particular solution in this case.

Annotations from class

In 1.4 and 1.5 we’ll learn the steps to obtain P(t) = Cekt from the ODE $\dfrac{dP}{dt} = kP,$ but for now notice that we can check: P(t) = Cekt means P′(t) = ekt ⋅ k by the chain rule, and so we see in fact P′(t) = kP(t) is satisfied.

More vocabulary


This general solution to a first order ODE $\dfrac{dP}{dt} = (\ln 2)P$ is a one-parameter family of functions P(t) = Cetln 2 indexed by C.

Annotations from class

Substituting P(0) = 1000 into P(t) = Cekt, we obtain 1000 = Cek ⋅ 0 = Ce0 = C, and so P(t) = 1000ekt.
Then, P(1) = 2000 yields 2000 = 1000ek ⋅ 1 ⟹ 2 = ek ⟹ k = ln (2), and so P(t) = 1000etln (2) is the particular solution in this case.

More vocabulary

A DE “typically" has infinitely many solutions; an initial value problem “typically" has one particular solution. An initial value problem consists of a DE along with initial condition(s).

Annotations from class

For example, the DE $\dfrac{dP}{dt} = (\ln 2)P$ has infinitely many solutions given by P(t) = Cetln 2 for any constant C.
On the other hand, the initial value problem $\dfrac{dP}{dt} = (\ln 2)P, \quad P(0) = 1000$ only has one solution P(t) = 1000etln 2.

Different “rates of change"

Instead of the “standard" time rate of change P′(t), we could consider the relative rate of change (that is, normalized by population) : $$\frac{P'(t)}{P(t)} = \frac 1 P \frac{dP}{dt}.$$

Annotations from class

This could be called the “fractional rate of change", and if we multiplied by 100 we’d call this the percent rate of change.
The simple population model is that the relative rate of change of population is constant, since this sentence translates to $$\dfrac{1}{P} \dfrac{dP}{dt} = k \iff \dfrac{dP}{dt} = kP.$$ This is another way to phrase the same model, earlier we said the rate of change of population $\dfrac{dP}{dt}$ was proportional to population.

Logistic population model

Let M > 0 represent the carrying capacity. The relative rate of change of P(t), the population at time t, is proportional to the “available space": $$\frac{1}{P} \frac{dP}{dt} = k(M-P) \qquad \mbox{ and so } \frac{dP}{dt} = kP(M-P)$$ for some constant k > 0.

Annotations from class

The motivation for this “refined model" is that our original simple population model predicted the population grows without bound — the example considered earlier P(t) = 1000etln 2 satisfied $\lim_{t \to \infty} \frac{dP}{dt} = \infty$. We know this isn’t realistic in the real world — eventually the habit will run out of space or resources and populations cannot increase without bound forever. We’ll see in sections 1.3, 1.4, 2.1 that this logistic model predicts limt → ∞P(t) = M provided P(0) > 0.

Example. A function y = g(x) is described by some geometric property of its graph. Write a differential equation of the form $\frac{dy}{dx} = f(x,y)$ having the function g as a solution.

  1. The slope of the graph of g at (x, y) is the sum of x and y.

  2. The line tangent to the graph of g at the point (x, y) intersects the x-axis at the point (x/2, 0).

  3. Every straight line normal to the graph of g passes through the point (0, 1).

Annotations from class

The key idea to keep in mind for deriving these first order ODE is that a first order ODE $$\dfrac{dy}{dx} = f(x,y)$$ is just a rule for the slope as a function of x and y.

  1. Translating the given rule, we have $$\dfrac{dy}{dx} = x + y.$$

  2. The slope $\dfrac{dy}{dx}$ at (x, y) must equal the slope between the points (x/2, 0) and (x, y), which we calculate as $\dfrac{y-0}{x-x/2} = \dfrac{y}{x/2} = \dfrac{2y}{x}.$ Therefore, we obtain the ODE $$\dfrac{dy}{dx} = \dfrac{2y}{x}.$$

Annotations from class

(c) Now, we have the slope $\dfrac{dy}{dx}$ at (x, y) must equal the negative reciprocal of the slope between the points (x, y) and (0, 1). The slope between those points is $\dfrac{y-1}{x-0} = \frac{y-1}{x},$ leading to the ODE $$\dfrac{dy}{dx} = - \dfrac{x}{y-1}.$$

Example. Show that the two-parameter family of functions y(x) = Acos (3x) + Bsin (3x) indexed by A, B solves the ODE y″ + 9y = 0.

Annotations from class

If y = Acos 3x + Bsin 3x, then y′ = −3Asin 3x + 3Bcos 3x and y″ = −9Acos 3x − 9Bsin 3x. Substituting yields $$-9A \cos 3x - 9B \sin 3x + 9(A \cos 3x + B \sin 3x) \overset{?}{=} 0 \implies 0 \overset{\checkmark}{=} 0,$$ and so y = Acos 3x + Bsin 3x solves the ODE for all x. Notice that this equation is second order, which corresponds to the general solution having 2 arbitrary constants (as we’ll discuss on the next slide).

Some more vocabulary

The normal form of an nth order ODE is y(n) = G(x, y, y′, ..., y(n − 1)). Typically, the general solution will be an n-parameter family of solutions, indexed by n “integration constants".

Annotations from class

Intuitively, each derivative we need to undo introduces one integration constant, so if the ODE involves the nth derivative, we expect the general solution to have n arbitrary constants.
